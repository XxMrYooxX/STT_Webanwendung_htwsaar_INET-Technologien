STT Engine - KIB-INET Projekt



Speech to Text Engine zum automatisierten Erstellen von Transskripts von Audio-Aufnahmen.
Upload einer unterstützten Audio-File über den Browse Button.
Der von Deepspeech erkannte Text wird in unten stehende Textbox ausgegeben.



Akzeptierte Formate / Codecs:
  • WAV Audio File
  • 16 bit
  • 16KHz MONO
